El programa debe ejecutarse desde la terminal con el siguiente comando:

java -jar practicarun.jar

A partir de ahí, el programa imprime por pantalla las instrucciones a seguir para hacer consultas

También se encuentra el .java original que hemos usado para hacer el ejecutable, por si se desea consultar.

REQUISITOS:

Es imprescindible tener instalada la última versión de java( java 14) para que se pueda ejecutar el programa.
Es necesario que en la misma carpeta donde está el ejecutable se encuentre el archivo de datos .nt
