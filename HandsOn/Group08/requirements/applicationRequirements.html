Application Requirements

Nuestro dataset contiene los puntos de recarga de vehículos eléctricos instalados en los edificios municipales de la comunidad de Madrid.

User interface

  • La aplicación permitiría al usuario filtrar los puntos de recarga de distintas formas(calle, distrito...).
  • La aplicación tendría un apartado de favoritos para permitir al usuario guardar sus puntos de recarga preferidos.
  • La aplicación estaría disponible tanto para Android como IOS
