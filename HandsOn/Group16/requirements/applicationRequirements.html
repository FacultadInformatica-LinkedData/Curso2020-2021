Application requirements document

User interface mock-up

The following interface is a preliminary schema for the future development of the application.

  1. samplingPoints () :

    Lists the whole set of codes of sampling points.

  2. magnitudeCodes () :

    Lists the whole set of codes of every magnitude.

  3. map () :

    Graph all sampling points coordinates into a map.

  4. graphMagnitude (magnitudeCode) :

    Graphs all the data available for the magnitude desired.

  5. graphDay (day, samplingCode) :

    Graphs all the data available in a sampling point given a specific day.

  6. yearlyTrends (magnitudeCode) :

    Graps the yearly mean of a magnitude given in all the sampling points.
