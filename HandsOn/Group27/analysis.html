Define Resource Naming Strategy

Methodology followed
We have separated the endpoints in resources and ontologies. The base domain of the chosen URI is http://api.zonasverdesmadrid.es. We have three classes: districts, parks, species.
1. From the districts class we have selected the following properties that relate a district from the datasets: hasId, hasName, hasTotal, hasPark, hasHeightAverage, hasYoung, justPlanted, hasOld, hasMature and hasOther.
2. From the parks class that is linked to the districts class we have introduced the following properties: hasName and hasSpecies.
3. From the species class that is linked to the districts and parks classes we have introduced the following properties: hasName and hasUnits.
