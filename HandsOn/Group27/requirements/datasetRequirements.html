Dataset Requirements

R1. Data are in one of the selected smart city domains.

  • Los datos pertenecen a la plataforma de datos abiertos de Madrid en concreto el conjunto de datos de Arbolado e parques y zonas verdes de Madrid

R2. Data are available as a CSV file.

  • En el apartado de descargas del conjunto de datos de Arbolado e parques y zonas verdes de Madrid podemos encontrar los datos en formato CSV y XLSX

R3. Data have an open license so they can be published.

  • La licencia de los datos que hemos escogido nos permite la reutilizacion de estos datos ya sean para fines comerciales o no comerciales, siempre y cuando se mencione como origen de los datos a la comunidad de Madrid.
    Información mas detallada aquí

R4. Data can be easily linked with generic real-world entities (e.g., locations)

  • Podemos linkar datos de especies de árboles, distritos... con fuentes da datos externas en dbpedia
    • Robinia pseudoacacia -> http://dbpedia.org/page/Robinia_pseudoacacia
    • Barajas -> http://dbpedia.org/page/Barajas_(Madrid)

R5. Documentation exists for the data (optional).

  • No hemos encontrado documentación adjunta a estos conjuntos de datos

R6. Data can come from multiple data sources (optional).

  • Nuestro dataset utilizará varios CSVs de el apartado de conjunto de datos de Arbolado e parques y zonas verdes de Madrid y si es necesario implementaremos datos de otras fuentes como dbpedia
