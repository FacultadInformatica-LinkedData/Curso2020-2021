Data Analysis:

Expedient Number String It is a string data type composed by letters and numbers to identify the exact accident.
Date DateTime It provides information about the date and hour when the accident was produced with format dd/mm/yyyyThh:mmssZ.
HourInterval String It provides information about the hour of the accident in hour intervals.
Address1 String It contains the street where the accident was produced.
Address2 String It contains the second street where the accident was produced in case it was in a crossing. Blank if it did not happen in a crossing.
Number String It can be a number, a number plus a letter or an empty value. It is the number of the street where the accident was produced.
District String It provides additional information so as to locate faster the accident.
Type of accident String It indicates the type of accident and some caracteristics.
Weather String It provides information about which was the weather when the accident occured.
Type of vehicle String It indicates the generic type of vehicle of the person who was affected by the accident.
Type of person String It indicates the type of person who was affected by the accident.
Age range String It is the age of the person involved in the accident. It is not an exact numer. It is a range of some ages. If blank, it is unknown
Sex String It provides information about the sex of the affected by the accident: male, female or blank (unknown).
Injury String It describes the grade of the injury that the person has after the accident. Empty values are equivalent to unknown information.

Benefits of linking with other datasets:

It offers a great utility for creating and delivering library data due to this data being sharable, extensible, and easily re-usable. In our case, reconciling the column “District” of our dataset with Wikidata, we can obtain, through SPARQL federated queries to WikiData's endpoint, the instances and properties that are not in our dataset and can be useful for our data (zip codes, coordinates, images of each district...).
