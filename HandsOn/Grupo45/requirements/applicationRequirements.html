Application Requirements

1. La aplicación podría utilizarse para automatizar un protocolo anticontaminación.

2. También se podría usar para hacer estudios medioambientales o de cualquier otra índole.

3. Se necesitaría una IA que trajese todos los datos de la base de datos monitorizada cada vez que llegasen novedades.

4. Otras aplicaciones como openRefine podrían ser también de utilidad.
