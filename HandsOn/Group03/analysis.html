Analysis:

 

1. Analyse Data Set

El conjunto de datos presenta todos los datos básicos de cualquier biblioteca pública de Castilla-La Mancha. Concretamente:

- Nombre de la unidad administrativa a la que pertenece (string-name)

- Código Biblioteca (integer)

- Nidem Biblioteca (string ó integer-integer)-

- Nombre biblioteca (string)

- Dirección (address)

- Código postal (integer)

- Municipio (string - name)

- Provincia (string- name)

- Teléfono(integer)

- Correo electrónico (email)

- Tipo de servicio, descripción (string) 

2. Analyse Licensing of the Data Source

El autor es la Junta de Comunidades de Castilla-La Mancha.

La licencia es de tipo Creative Commons Attribution Share-Alike

La licencia puede mantenerse una vez trabajo en el dataset.

3. Resource Naming Strategy

Al ser un trabajo sobre la comunidad autónoma de Castilla-La Mancha, puede ser atractivo utilizar una URI que proporcione información desde la raíz, es decir, por ejemplo:

<www.example.org#CastillaLaMancha#>

A partir de ahí, podremos ir nombrando todas las entidades que identifiquemos en el dataset, al igual que las propiedades y sus clases.
