Application requirements:

1) Las especificaciones técnicas serían las necesarias para poder lanzarla a la App Store y Play Store, con disponibilidad para cualquier sistema operativo móvil.

2) La aplicación podría tener un espacio para poder recoger información de forma segura y confidencial.

3) Es recomendable que la aplicación disponga de un visor de datos para que los usuarios puedan comparar con otras zonas del país.

4) Funcionalidades extra como recordatorio de ejercicio (opcional)
