ANALYSIS

Potential license of the dataset to be generated

As we have extracted the datasets from the website of the Madrid Autonomous Community, the most restrictive license you should have is that of this source. In the page they indicate that the license is the following one: CC BY 4.0. This allows us a commercial or non-commercial use of the datasets. Therefore we could offer the same license for others to take advantage of the data as we have already done, or put a more restrictive license that for example does not allow a commercial use.

Resource naming strategy

We have followed a simple naming strategy, with a name directly related to the resource.

Property Description
centerCod Unique code for each centre
centerTypeCode Unique code for each type of centre
description Description of the center (E.G: primary school). Data is in Spanish
titularity Who is the owner of the institution
hasEmail email address of the centre
fax Contact fax of the centre
hasTelephone Contact phone of the centre
webpage Webpage of the centre
cityName City where the centre is located
cityCod Unique code for each city
districtName District (when applicable) where the centre is located
districtCod Unique code for each district
longitude Component longitude of the centre coordinates
latitude Component latitude of the centre coordinates
street-address Street address of the centre
postal-code Postal code of the centre
activityCod Unique code for each extracurricular activity
takePlace Centres where the activity is offered
