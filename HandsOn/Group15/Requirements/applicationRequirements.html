Application Requirements

Application Description

This application allows users to search education centers and the after-school activities they offer in Madrid.

The user interface will have the following requirements:

  • An exclusive window for the application will be displayed when it is executed
  • It will have a map of the Community of Madrid
  • The user will have two drop-down menus to select the municipality and type of centre
  • The list of centres you return as a result will be displayed in a separate window from the previous one

Application Requirements

  • The implementation of the application will be in Java
  • The user interface will be in Java
  • The application will be a desktop app for Windows (WPF)

Resource naming strategy

In general terms, the names that have been chosen are those that are standard and typical of the general specification. In cases where these names have not been chosen (mainly because we have had to create them ourselves) other names have been used which make it easier to identify them unambiguously.

Main bennefits of this app

The main benefit of this application will be that future users will be able to find all the necessary information to your children go to a centre with the characteristics they are looking for without having to search through its pages or move (through the pandemic in which we find ourselves is very difficult).

The main benefit is that we can find centrally all the information to make the best decision.
