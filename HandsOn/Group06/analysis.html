► Tabla con las columnas del CSV

Columna Tipo de variable Descripción
Código interno int Código de identificación del contenedor
Tipo string Tipo del residuo (vidrio, papel, orgánico...)
Descripción Modelo string Modelo del contenedor
Cantidad int Número de contenedores en el situado
Lote int Lote al que pertenece (LOTES 1,2 o 3)
Distrito string Distrito en el que se encuentra
Barrio string Barrio en el que se encuentra
Tipo vía string Clase de vía (calle, plaza, etc.)
Nombre string Denamonación de la vía
Número int Número de la vía
Coordenada X float Situación del objeto en su coordenada X en medida UTM (en centímetros).
Coordenada Y float Situación del objeto en su coordenada Y en medida UTM (en centímetros).
Latitud float Latitud del objeto
Longitud float Longitud del objeto
Dirección string Los campos tipo vía, nombre y número en uno solo
