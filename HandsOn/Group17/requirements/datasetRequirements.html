Dataset Requirements

REQUISITOS

  • R1. Los datos están en uno de los dominios seleccionados de la ciudad inteligente
  • R2. Los datos estan disponibles como un fichero CSV
  • R3. Los datos tienen una licencia abierta para que puedan ser publicados
  • R4. Los datos pueden vincularse fácilmente con entidades genéricas del mundo real (por ejemplo, las ubicaciones)
  • R5. Existe documentación para los datos (opcional)
  • R6. Los datos pueden provenir de múltiples fuentes de datos (opcional)
